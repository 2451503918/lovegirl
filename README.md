# LikeGirl - LGNewUi

情侣小站全新改版，全面升级为 LGNewUi 新版样式！

## ✨ 更新亮点

- 「lgnewui」视觉风格统一：首页、点滴、留言、轨迹、相册、清单、关于、文章详情页等所有页面均已更新为新版视觉风格
- 「lgnewui-message-card」留言卡片：全新设计的留言界面，包含统计展示、卡片列表、表情选择等功能
- 「lgnewui-article-detail-card」文章详情页：美观的文章阅读体验，包含导航、作者信息展示

## 📝 使用说明

### 食用方法

1. 打开根目录下的 `admin` 文件夹
2. 找到 `Config_DB.php` 文件，根据注释提示配置数据库相关信息
3. 请认真填写安全码，尽量设置的复杂一些，修改密码等敏感信息需输入安全码
4. `love_db.sql` 为 SQL 文件，导入到数据库中即可
5. 默认账号密码：`admin` / `love2025`

### 文章书写

文章书写需了解 HTML 标签基本语法。可以参考默认文章的格式进行创作。
